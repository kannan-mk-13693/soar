Playbook Data Audit — DB to ElasticSearch

Overview

Records a single playbook block-execution event into the audit trail. Called once per block (or per status change) by PlaybookAction.addExecutionAudit(...) in the ADSF executor layer, delegated through the PlaybookFacade SPI, and finally written to Elasticsearch by the ELA-side implementation in ELAIncidentWFHandler.

Parameters

Parameter Resolves To Description
IM_WF_INTERFACE FQCN of SPI impl Used by PlaybookFacade static block to bootstrap the delegate.
PLAYBOOKAUDITINDEX DAEUtils.IndexType ES index family for playbook audit records.
DEFAULT_PLAYBOOK_AUDIT_NAME_PREFIX "playbookaudit_" Index name prefix; suffixed by date bucket derived from MODIFIED_TIME.
